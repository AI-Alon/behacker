Fat GET Poisoning

Smuggling a body into a GET request to override the cached response

Some servers allow a GET request to carry a body, and use body parameters over query parameters. The cache, however, keys only on the URL. Inject a malicious search parameter in the body to poison the cached GET /search?q=normal response.
