Cache Key Manipulation

Web Cache Poisoning via Unkeyed Headers

The cache key is based only on the URL path. But the server reflects X-Forwarded-Host into the response (e.g. for generating absolute URLs). Inject a malicious host to poison the cached page — every subsequent visitor receives the poisoned response.
