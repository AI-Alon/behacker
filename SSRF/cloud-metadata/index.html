Cloud Metadata SSRF

Server-Side Request Forgery — AWS/GCP/Azure Metadata

Cloud instances expose a metadata API at 169.254.169.254 that returns IAM credentials, instance info, and user-data scripts. An SSRF vulnerability lets an attacker hit this endpoint and steal cloud credentials.
